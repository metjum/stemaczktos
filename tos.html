Introduction

This extension has a single purpose: to convert Euro prices displayed on the Steam store to Czech Koruna.

Functionality

The extension searches for all instances of Euro prices on the page and replaces them with their equivalent values in CZK based on the current exchange rate. The conversion process is seamless and provides users with a better understanding of prices in their local currency.

Permissions

Storage Justification

The extension uses the 'storage' permission to save the current exchange rate for Euro to Czech Koruna. This allows the extension to efficiently retrieve and apply the exchange rate for price conversions without needing to fetch it from an external source each time the page loads.

Active Tab Justification

The 'activeTab' permission is necessary to allow the extension to access the currently active tab where the user is browsing the Steam store. This permission enables the extension to modify the content of the page and replace Euro prices with their equivalent values in CZK.

Scripting Justification

The 'scripting' permission is required for the extension to run JavaScript code that modifies the DOM of the Steam store pages. This allows the extension to search for price elements and replace their content dynamically as users navigate the store.

Host Permission Justification

The host permission for 'store.steampowered.com' is necessary because the extension specifically targets this domain to perform price conversions. By having this permission, the extension can access the HTML content of the Steam store pages and accurately replace Euro prices with their equivalent values in CZK.
